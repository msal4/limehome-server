# limehome server

## Run

- Install dependencies

```bash
yarn
```

- Set environment variables

```bash
cp .env.example .env
```

set the required `DATABASE_URL`

and run the server

```bash
yarn start
```

## Public

The server is hosted in my private dokku (basically an open source heroku) server.
